Set up Sentry and enable Sentry integration.

This function is a convenience for users that may not want to use a lambda to configure Sentry. As there are so many options, not everything is included here - please use init if you need to change any other settings.

We recommend using keyword arguments for this function. Every single parameter has a default value that should match Sentry's default setup.
