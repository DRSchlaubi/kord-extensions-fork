Paginator

open class Paginator(bot: ExtensibleBot, channel: MessageChannelBehavior, name: String, pages: List<String>, owner: User?, timeout: Long, keepEmbed: Boolean)
Content copied to clipboard

Interactive embed with multiple pages using emoji reactions as user inputs.

NOTE : .send() needs to be called in order for the paginator to be displayed.

Parameters

bot

Current instance of the bot.

channel

Channel to send the embed to.

keepEmbed

Keep the embed and only remove the reaction when the paginator is destroyed.

name

Title of the embed.

owner

Only user capable of interacting with the paginator. Everyone is able to if it is set to null.

pages

List of the embed pages.

timeout

Milliseconds before automatically deleting the embed. Set to a negative value to disable it.
