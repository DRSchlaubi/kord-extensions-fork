      • getUrlRegex

         final Regex getUrlRegex()

        Regular expression used to extract domains from messages.

        If you customize this, it must contain exactly one capturing group, containing the full domain name, and optionally the path, if this is an actual URL. You can mark a group as non-capturing by prefixing it with ?:.

        The provided regex comes from https://urlregex.com/ - but you can provide a different regex if you need detection to be more sensitive than just clickable links.
